# mydwm

dwm with patches for a better experience.

## Patches
- vanitygaps
- pertag
- movestack
- attachbelow
- actualfullscreen
- swallow
- noborder
- focusonclick
- warp
- autostart
- restartsig
- scratchpad
- cyclelayouts
- savefloats
- statusallmons
- systray
- ipc

## Dependencies
- libx11
- libxft
- libxinerama
- yajl
- jq
- xorg-xprop
- picom
- dunst
- feh
- flameshot
- polybar

## Install
```bash
git clone https://github.com/Saad-Dev-8/mydwm
cd mydwm
sudo make clean install
```
