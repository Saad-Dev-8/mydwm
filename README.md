# mydwm

dwm with patches for a better experience.

## Patches
- vanitygaps
- pertag
- movestack
- attachbelow
- actualfullscreen
- swallow
- noborder
- focusonclick
- warp
- autostart
- restartsig
- scratchpad
- cyclelayouts
- savefloats
- statusallmons
- systray
- ipc

## Dependencies
- libx11
- libxft
- libxinerama
- yajl
- jq
- xorg-xprop
- picom
- dunst
- feh
- flameshot
- polybar

## Install
You can use install script with:
```bash
cd mydwm
chmod +x install.sh
./install.sh
```
